# 记事簿

独立记事应用，基于悠厨项目模板搭建，功能依据《记事簿 app》需求文档实现。

## 功能模块

1. **基础记事** — 快速/完整编辑、文本/清单/多媒体、模板、自动保存、预览
2. **记事管理** — 首页列表、分类、标签、搜索筛选、星标收藏、回收站
3. **同步与隐私** — 同步管理、离线模式、隐私模式、备份导入导出
4. **提醒** — 定时/重复/倒计时/位置提醒、提醒管理与详情
5. **账号与设置** — 登录注册、主题切换、密码修改、个人统计

## 使用

```bash
npm install
npm run dev
```

或双击 `打开记事簿.bat`（会先 build，再通过本地预览服务器打开）。

单机本地运行，无需登录。首次使用有引导页，可在「我的 → 使用引导」中重新查看。
